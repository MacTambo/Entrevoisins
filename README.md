Clone this repository and import into **Android Studio**
```bash
https://github.com/MacTambo/Entrevoisins.git
```

## Compile and execute
From Android Studio:
1. ***Build*** menu
2. ***Rebuild project***
3. ***Select app and your emulator or phone***
4. ***Click on the play button***
